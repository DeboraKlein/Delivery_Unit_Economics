# Delivery_Unit_Economics
Pipeline ETL em Databricks/Delta Lake para análise de Unit Economics (UE) de um Marketplace Two-Sided de entregas. Unifica Pedidos, Pagamentos e Logística para calcular métricas de P&amp;L, como Lucro Bruto Unitário. A Arquitetura de Medalhões (Bronze/Silver/Gold) cria um modelo dimensional otimizado para consumo em BI (Power BI).
